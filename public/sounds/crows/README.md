# Crow/Corvid Deterrent Sounds

Place MP3 files here to deter crows, magpies, jackdaws, and similar corvids. Effective sounds include:

- Hawk/raptor calls
- Owl hoots
- Crow distress calls
- Predator alarm sounds
- Human voices
